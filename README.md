Прошивка для Arduino UNO с USB Host Shield для зарядки DualShock 3 без ПК/консоли.<br><br>
<img width="936" height="741" alt="case" src="https://github.com/user-attachments/assets/c5edc7f7-1a50-41ec-9845-ed3806ae9c9c" /><br><br>
DualShock 3 не заряжается от обычных источников питания из-за отсутствия сигналов данных (D+/D-) на кабеле USB, необходимых для «рукопожатия» контроллера с источником питания.<br>
Для работы USB Host Shield нужно спаять отмеченные на фото перемычки.<br><br>
<img width="1280" height="963" alt="jumpers" src="https://github.com/user-attachments/assets/f5f407ef-dd46-444a-9f9b-7182b45d04c3" /><br><br>
Используйте mini-usb кабель с возможностью передавать данные.<br>
Так же Arduino UNO с USB Host Shield чувствительна к источнику и кабелю питания, я чудом подобрал комбинацию из вороха своих зарядок.<br>
